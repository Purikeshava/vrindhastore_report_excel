# 📊 Vrinda Store Sales Dashboard (Excel)

## 📌 Overview

This project is an interactive Excel Sales Dashboard developed to analyze the sales performance of Vrinda Store for the year 2022. I cleaned and organized the raw sales data, created Pivot Tables and Pivot Charts, and designed a dynamic dashboard using slicers for easy filtering. The dashboard provides valuable insights into sales trends, customer behavior, revenue distribution, order status, and sales channel performance, helping users make data-driven decisions.

## 🛠 Skills Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers
- IF Functions
- Conditional Formatting
- Data Visualization
- Dashboard Design

## 📈 Key Insights

- Women generated a higher percentage of sales than men.
- Adults placed the highest number of orders among all age groups.
- Most orders were successfully delivered.
- Bengaluru, Hyderabad, Chennai, New Delhi, and Mumbai were among the top revenue-generating cities.
- Amazon and Myntra contributed significantly to overall sales.
